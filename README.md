# 📚 Blog List App (React JS)

A simple Blog List application built using **React JS**. This project displays a list of blogs with title, description, and published date in a clean UI.

---

## 🚀 Features

- Display list of blogs dynamically
- Reusable components (BlogList & BlogItem)
- Clean and responsive UI
- Navigation using React Router
- Pages: Home, About, Contact, Not Found

---
## 🧩 Components Overview

### 1. Header
Contains navigation links for Home, About, and Contact pages.

### 2. Home
Displays user profile and list of blogs.

### 3. BlogList
Receives blogs data as props and renders multiple blog items.

### 4. BlogItem
Displays individual blog details like title, description, and published date.


## 🛠️ Tech Stack

- React JS
- JavaScript (ES6)
- React Router DOM
- CSS
